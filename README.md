# Beispiel Programm for jspm

## prerequisites

* node.js
* npm install -g jspm bower serve

## install jspm

	npm install -g jspm
	jspm install
	jspm install jquery
	jspm install npm:lodash
	
	//type script plugin
	jspm install ts
	
## install bower polymer
	bower update
	
## Run in browser

	npm install -g serve
 	serve
	chrome htpp://localhost:3000
	
## distribute
	jspm build (error in windows)
